# Prime-# Prime Balance Spiral

This project explores a new structural interpretation of prime numbers.

## Key Idea

Prime numbers are characterized as numbers that cannot be generated by:

a + b + c + d = ab - cd

## Results

- All odd primes up to 100,000 are outside the balance structure

- Almost all odd composite numbers are inside

- Exception: 1, 9

## Implications

- New viewpoint on prime detection

- Structural explanation of primes

- Potential link to infinitude of primes
